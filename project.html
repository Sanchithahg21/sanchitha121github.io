2. Secure Data Transmission

The project titled “Secure Data Transmission” is designed for providing security during transmission of data. In order to provide security of sensitive data store in public clouds, a commonly used approach is to encrypt data before uploading into public clouds. So that to provide confidentially of stored public cloud data, the encryption mechanism should also be able to support the access of confidential data. In this project, we propose public key encryption schema for generation of secret key and encrypt the data using that key

Technology Used

Java,HTML,CSS
